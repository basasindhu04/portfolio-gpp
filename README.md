# Basa Sindhu Latha - Portfolio

## Project Description
A fully responsive, animated personal portfolio website built to showcase my backend engineering and development skills. Features advanced CSS effects, on-scroll animations, and a responsive design across all breakpoints.

## Tech Stack
- Next.js (React)
- Tailwind CSS v4
- Framer Motion
- TypeScript
- Radix UI

## Local Setup Instructions
To run this project locally, follow these steps:

1. Clone the repository:
   ```bash
   git clone https://github.com/basasindhu04/portifolio.git
   ```
2. Navigate into the directory:
   ```bash
   cd portifolio
   ```
3. Install dependencies:
   ```bash
   npm install
   # or
   pnpm install
   ```
4. Run the development server:
   ```bash
   npm run dev
   # or
   pnpm dev
   ```
5. Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
